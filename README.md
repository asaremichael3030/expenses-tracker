# expenses-tracker
A simple and responsive Expense Tracker Web App built with HTML, CSS, and JavaScript to help users manage their daily income and expenses efficiently.

🚀 Features
➕ Add income and expense transactions
➖ Delete transactions easily
📊 Real-time balance calculation
💵 Tracks total income and expenses separately
💾 Data stored in browser using Local Storage
📱 Responsive design (works on mobile & desktop)

🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Functionality and logic
Local Storage API – Data persistence

⚙️ How to Use
Clone the repository:
git clone https://github.com/asaremichael3030/expense-tracker.git
Navigate to the project folder:
cd expense-tracker
Open index.html in your browser

Live demo
https://expensestracker11.netlify.app/

✨ How It Works
Enter a description and amount
Positive amount → Income
Negative amount → Expense
Click Add Transaction
Your balance updates instantly
Data is saved automatically in your browser
